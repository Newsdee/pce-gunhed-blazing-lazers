# Gunhed - Blazing Lazers

Gunhed is a fantastic PC Engine shmup, but it was notorious for having sound mixed at a low volume.

This ROM hack modifies the sound volume of music, sound effects, and sound samples,
so that the game can be enjoyed at similar volume levels as other games in the PC Engine library.

![Title Screen](screenshots/Gunhed (Japan)_000.png)
![Gameplay](screenshots/Gunhed (Japan)_001.png)
![Sound test](screenshots/Gunhed (Japan)_002.png)

The file also contains an optional patch to make the ROM start directly on the sound test menu.


# How to use this patch
To use this patch, you will need the original ROM file for PC Engine, and use a BPS patcher to apply it.
This will give you a new file that you can use with a Turbo Everdrive for real hardware, or an emulator such as [Mesen2](https://github.com/SourMesen/Mesen2)

You can find an online patcher at [ROMHacking.net](https://www.romhacking.net/patch/).
