- **DDA voice streaming** (bank `$00`, ROM `$EA40`): `LDA ($40,X) / STA $0806` loop

  - Nibble-packed PCM: `AND #$0F / ASL` (lower) and `AND #$F0 / LSR / LSR / LSR` (upper)

## DDA Voice Sample Note

Gunhed is a **HuCard** — there is no CD-ROM² ADPCM hardware (`$0D00–$0D0F`).
Voice samples use PSG **DDA mode** (R4 bit 6 = 1, chON=1) streamed through R6 (`$0806`).
The sample bank is assigned at runtime via ZP table `$4C,X`; static analysis cannot
determine which ROM bank(s) hold sample data without emulator tracing.
